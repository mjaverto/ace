# Contributing

Thanks for your interest. The full contributor guide — including the recipe for adding a new source plugin, dev setup, fixture conventions, and schema-drift discipline — lives at [`docs/contributing.md`](docs/contributing.md). Start there.

## Code of Conduct

Be kind. Assume good faith. A formal Code of Conduct will be added before 1.0; until then, the maintainer's judgment applies.

## Issues / PR templates

Not provided in v0. Open an issue or PR with whatever shape feels right; the maintainers will help format it.
